# 🧮 活動量等級與對應係數（TDEE 估算參考）

活動量決定你每日總能量消耗（TDEE）的關鍵，是以基礎代謝率（BMR）乘上活動係數計算。

---

## 📊 活動量等級對照表

| 活動量等級        | 描述                                   | 係數 (TDEE = BMR × 係數) | 適合對象範例               |
|-------------------|----------------------------------------|----------------------------|----------------------------|
| 🛋️ 久坐型（Sedentary）     | 幾乎不運動，整天坐著辦公                     | `1.2`                      | 辦公族、靜態生活者           |
| 🚶‍♂️ 輕度活動（Lightly Active） | 每週運動 1–3 天，強度輕，如快走、瑜伽              | `1.375`                    | 輕度訓練、通勤常步行者        |
| 🏃‍♂️ 中度活動（Moderately Active） | 每週運動 3–5 天，中等強度訓練                    | `1.55`                     | 健身房訓練、有氧/重訓結合者     |
| 🧗‍♂️ 高活動（Very Active）     | 幾乎每日訓練或中高強度勞力工作                    | `1.725`                    | 體育教師、重訓族、建築工人等     |
| 🏋️‍♀️ 超高活動（Super Active）   | 每日高強度訓練或雙倍訓練                         | `1.9`                      | 運動員、備賽選手、軍訓訓練中     |

---

## ✅ 如何自評你的活動量？

請依以下幾點自我評估活動等級：
- 每週運動頻率？（幾天）
- 每次運動強度？（會不會喘、流汗）
- 每日步行/站立時間？（NEAT 活動）
- 是否做阻力訓練（重訓）或高強度間歇（HIIT）

---

## 📎 使用範例

如果你的 BMR 是 1650 kcal，活動量為「中度活動」：

TDEE = 1650 × 1.5 = 2475 kcal（維持體重）

若你目標是減脂，可設定：

每日攝取熱量 = TDEE − 300 = 約 2175 kcal

| 目標             | 建議熱量             |
|------------------|----------------------|
| ✅ 穩定減脂       | TDEE − 300 ≈ **2175 kcal**（推薦） |
| 💪 若偏向增肌為主 | TDEE − 100～200 ≈ **2375–2275 kcal** |
| 🧨 激進減脂       | TDEE − 500 ≈ **1975 kcal**（⚠️ 易掉肌） |

# 瘦身方案：練出馬甲線

> **飲食建議**  
> - 熱量：每日 1500–1800 kcal（運動量較大時可接近 2000 kcal）  
> - 三大營養素比例：蛋白質 30%／碳水化合物 40%／脂肪 30%  
> - 優先攝取蛋白質：雞胸、魚、蛋白粉、豆腐等  
> - 多吃高纖蔬菜，控糖少油  

---

## 1. TABATA 徒手訓練方案（居家＋無器械）

**頻率**  
- 每週 4 次（隔天訓練）  
- 每次 20–25 分鐘（不含熱身 & 放鬆）

**流程**  
1. **熱身（5 分鐘）**  
   - 原地小跑／開合跳／髖關節活動  
2. **TABATA 循環**  
   - 20 秒高強度 + 10 秒休息，共 8 輪（4 分鐘）  
   - 完成一組後休息 1 分鐘，再進行下一組  
   - 共 3 組不同動作  
3. **核心收尾（4 分鐘）**  
   1. 平板撐 60 秒  
   2. 側平板（左右）各 30 秒  
   3. 仰臥抬腿 30 秒 + 休息 15 秒 ×2  
4. **放鬆拉伸（3 分鐘）**

| 組別 | 動作示例            | 要點                                                 |
| ---- | ------------------- | ---------------------------------------------------- |
| 組 1  | 高抬腿跑            | 腰背挺直，收緊核心，用腹部發力帶動雙腿抬高           |
| 組 2  | 俯臥撐→改良開合跳    | 無力時可改跪姿俯臥撐＋小跳分腿                       |
| 組 3  | 登山跑 + 仰臥扭轉    | 登山跑時核心收緊；仰臥扭轉要帶動腹斜肌               |

> **進階選項**：  
> - 每組增加至 5 輪  
> - 將間歇改為 30 秒做 + 15 秒休息

## 2. 健身房器械＋自由重量方案

**頻率**  
- 每週 4–5 次：3 次力量訓練 + 2 次有氧／核心

### 周計畫示例

| 星期 | 訓練內容                                |
| ---- | --------------------------------------- |
| 周一 | 推＋核心                                |
| 周二 | HIIT（跑步機／風扇車 25–30 分鐘）＋拉伸  |
| 周三 | 拉＋核心                                |
| 周四 | 休息或輕度有氧（快走／騎行 30 分鐘）     |
| 周五 | 下肢＋核心                              |
| 周六 | 自選有氧（橢圓機／跳繩 25–30 分鐘）      |
| 周日 | 休息                                    |

### 力量訓練＋核心流程

1. **大肌群力量**  
   - 每動作 3–4 組 × 8–12 次，組間休息 60–90 秒  
   - 推：臥推／上斜啞鈴／俯臥撐  
   - 拉：高位下拉／單臂啞鈴划船／面拉  
   - 下肢：深蹲／硬舉／腿舉機  

2. **核心專項**  
   - 3 組 × 12–15 次（側平板 30–45 秒）  
     - 懸空卷腹（Hanging Leg Raise）  
     - 俄羅斯轉體（可持小啞鈴）  
     - Cable Crunch（滑輪下壓卷腹）  
     - 側平板＋臀橋交替  

3. **有氧衝擊**（周二／周六擇一）  
   - 跑步機：1 分鐘快跑 / 1 分鐘慢走，交替 10–12 輪  
   - 或 風扇車 / 划船機 同樣進行間歇式 HIIT  

4. **放鬆拉伸（5–10 分鐘）**  
   - 針對胸、肩、背、髖腿、腰腹等部位拉伸  

---

## 額外建議

- **漸進超負荷**：每週或兩週提升訓練重量或組數  
- **充足睡眠**：7–8 小時，促進恢復  
- **補水**：訓練前中後持續補水，每天 ≥2.5 L  
- **監測進展**：每兩週拍照／測量腰圍／記錄體重與體脂  

> **堅持 8–12 週**，你會明顯感受到腰腹線條收緊，馬甲線漸現！加油！  

# 💪 增肌減脂訓練計畫（BMI 28 ➝ 25）

目標：透過規律訓練與飲食控制，降低體脂、維持肌肉量，達到健康身材。

---

## 🧍‍♂️ 基本資料與目標

- **目前 BMI**：28  
- **目標 BMI**：25 以下  
- **基礎代謝率 (BMR)**：1650 kcal  
- **總消耗熱量 (TDEE)**：約 2450 kcal  
- **每日建議熱量攝取**：2150 kcal（TDEE - 300）  
- **每日建議蛋白質攝取**：每公斤體重 1.6–2.2 克  

---

## 📅 訓練計畫總覽

每週訓練 5 天，每天擇一訓練方式：
- 🏠 居家：**TABATA 高強度間歇訓練**
- 🏋️‍♂️ 健身房：**Push / Pull / Legs 重量訓練**

| 星期   | 訓練內容           |
|--------|--------------------|
| 週一   | Push（胸肩三頭）     |
| 週二   | Pull（背二頭）       |
| 週三   | TABATA（全身）      |
| 週四   | Legs（腿臀）         |
| 週五   | TABATA（核心＋燃脂） |
| 週六   | 選擇性（輕有氧或休息） |
| 週日   | 休息                 |

## 🏠 居家 TABATA 訓練（20秒動作 / 10秒休息 × 8 組）

```
- 波比跳
- 登山者
- 跳躍深蹲
- 快速開合跳
```

### 🧘‍♀️ 回合 2：核心＋下肢

```
- 平板支撐轉體

- 俯臥撐（寬）
```

## 🏋️‍♂️ 健身房訓練（重量訓練）

### PUSH（胸＋肩＋三頭）

```
- 啞鈴肩推 3x10
```

### PULL（背＋二頭）

```
- 高位下拉 3x10
```

### LEGS（腿＋臀）

```
- 深蹲 or 腿推 4x10
- 腿彎舉 / 腿伸展 3x12
- 臀橋 or 臀推 3x10
- 小腿抬舉 3x15
```

## 🧠 附註與提醒

- 每次訓練加熱身（5–10 分）與收操（拉伸）
- TABATA 適合高燃脂期與無設備日
- 重訓重視動作正確，重量適中即可
- 可於重訓後加做 10–15 分有氧（如跑步、登階）

## 🍱 飲食建議（2150 kcal）

- 蛋白質：120–150g  
- 碳水化合物：180–220g  
- 脂肪：50–60g  
- 優先攝取：雞胸肉、豆腐、蛋、糙米、燕麥、地瓜、蔬菜  
- 訓後建議：乳清蛋白 + 水果（如香蕉）
